Tags para marcação de texto

Inicio

A linguagem HTML é uma linguagem de marcação de hipertexto. Basicamente isso significa que ela foi criada como uma linguagem para definção de formato do texto e a conexão entre diversas fontes ou links.

As principais tags para se trabalhar com texto são: h1 - h6, p, a, em, strong, pre e br. Além destas tags, também podemos contar com as tags sub, sup, cite, blockquote, além de uma míriade de outras tags que podem ser consultadas aqui.

Clique no nome de cada tag para ver mais.

tag h1 até h6

tag h1 até h6

As tags h1 - h6 são usadas para criar headings. É interessante respeitar a sequência dos headings, usando h1 para o conteúdo principal, h2 para subtitulos, e assim por diante. Para formatar os headings devemos utilizar css e classes. Ou seja, utilzar:

tag p

tag p

A tag p é utilizda para criação de parágrafos. Ela viabiliza a estruturação de textos de forma organizada, adicionando uma linha ao final de cada tag. Também adiciona espaços ao inicio e ao final de cada parágrafo.

tag pre

tag pre

A tag pre é semelhante a tag p, sendo utilizda para criação de textos, porém a tag pre respeita os espaços digitados no código html. Exemplo:

<pre> 
                            Este      texto
                                        tem espaços digitados de forma irregural
                                e a tag os mantém, conforme foram criados no HTML.
<pre>

Resulta em:

                        Este      texto
                                    tem espaços digitados de forma irregural
                            e a tag os mantém, conforme foram criados no HTML.

tags sub e sup

tag sub e sup

A tag sub é utilizda para criação de texto subscrito. Já a tab sup gera este efeito no texto. São particularmente úteis para notação científica e matemática.

<p>X<sup>2</sup> + 2X<sup>3</sup> ou H<sub>2</sub>O</p>

Gera o seguinte resultado:

X2 + 2X3 ou H2O

tags q e blockquote

tag blockquote

A tag blockquote é utilizada para marcar citações longas e a tag q para marcar citações curtas. Exemplo:

<blockquote cite="https://www.goodreads.com/work/quotes/24738161-computing-machinery-and-intelligence"> Let us return for a moment to Lady Lovelace's objection, which stated that the machine can only do what we tell it to do. One could say that a man can "inject" an idea into the machine, and that it will respond to a certain extent and then drop into quiescence, like a piano string struck by a hammer. Another simile would be an atomic pile of less than critical size: an injected idea is to correspond to a neutron entering the pile from without. Each such neutron will cause a certain disturbance which eventually dies away. If, however, the size of the pile is sufficiently increased, the disturbance caused by such an incoming neutron will very likely go on and on increasing until the whole pile is destroyed. Is there a corresponding phenomenon for minds, and is there one for machines? There does seem to be one for the human mind. The majority of them seem to be "sub critical," i.e. to correspond in this analogy to piles of sub-critical size. An idea presented to such a mind will on average give rise to less than one idea in reply. A smallish proportion are supercritical. An idea presented to such a mind may give rise to a whole "theory" consisting of secondary, tertiary and more remote ideas. Animals’ minds seem to be very definitely sub-critical. Adhering to this analogy we ask, "Can a machine be made to be super-critical? ― Alan Turing, <a href="https://www.goodreads.com/work/quotes/24738161-computing-machinery-and-intelligence">Computing machinery and intelligence</a> </blockquote> Como disse Turing: <q>"If a machine is expected to be infallible, it cannot also be intelligent."</q>

Tags para criação de listas

Inicio

O html permite criação de listas ordenadas, listas não ordenadas e listas descritivas. Clique em cada tag para entender como utilizar cada tipo de lista.

tag ol

A tag ol gera listas ordenadas, que são listas com índice de posição, definindo a posição do primeiro ao último elemento

Este código: <ol> <li>Primeiro item;</li> <li>Segundo item;</li> <li>terceiro item</li> </ol> Gera esta lista:

  1. Primeiro item;
  2. Segundo item;
  3. terceiro item;
tag ul

A tag ul gera listas desordenadas, que são listas sem índice de posição.

Este código: <ul> <li>Primeiro item;</li> <li>Segundo item;</li> <li>terceiro item</li> </ul> Gera esta lista:

  • Primeiro item;
  • Segundo item;
  • terceiro item;
tag dl

A tag dl gera listas descritivas, que são listas que contém descrições de tipos, como abaixo:

Este código: <dl> <dt>Café</dt> <dd>Bebida quente de cor escura, de origem vegetal.</dd> <dt>Leite</dt> <dd>Bebida de cor clara, de origem animal. Consumida quente ou gelada.</dd> <dt>Refrigerante</dt> <dd>Bebida artificial, consumida gelada</dd> </dl> Gera esta lista:

Café
Bebida quente de cor escura, de origem vegetal
Leite
Bebida de cor clara, de origem animal. Consumida quente ou gelada.
Refrigerante
Bebida artificial, consumida gelada
